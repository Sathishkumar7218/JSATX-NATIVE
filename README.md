# JSATX-NATIVE 🚀

JSATX-NATIVE is a reusable Java library designed to provide utility features and core functionalities that can be integrated into any Java or Spring Boot project.

---

## 📦 Installation

JSATX-NATIVE is distributed using **JitPack**, so you can easily add it as a Maven dependency.

### 1️⃣ Add JitPack Repository

Add the following inside your `pom.xml`:

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
<dependency>
    <groupId>com.github.Sathishkumar7218</groupId>
    <artifactId>JSATX-NATIVE</artifactId>
    <version>1.0.0</version>
</dependency>```
